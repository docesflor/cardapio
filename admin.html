Conexão com GitHub

Configure seu token de acesso para que as alterações sejam salvas automaticamente no repositório.

O token fica salvo só no seu navegador (localStorage). Nunca é enviado a terceiros.

COMO CRIAR O TOKEN NO GITHUB

  1. Acesse github.com → Settings → Developer settings
  2. Clique em Personal access tokens → Tokens (classic)
  3. Clique em Generate new token (classic)
  4. Dê um nome (ex: doces-flor-admin)
  5. Marque a permissão repo (acesso completo)
  6. Clique em Generate token e copie o valor
  7. Cole o token aqui acima e salve
